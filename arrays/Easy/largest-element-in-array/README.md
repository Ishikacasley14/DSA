# 1. Largest Element in Array

**Difficulty**: Basic

---

##  Problem Statement

Given an array `arr[]`. The task is to find the **largest element** and return it.

---

##  Examples

**Example 1**  
Input: `arr[] = [1, 8, 7, 56, 90]`  
Output: `90`

**Example 2**  
Input: `arr[] = [5, 5, 5, 5]`  
Output: `5`

**Example 3**  
Input: `arr[] = [10]`  
Output: `10`

---

##  Constraints

- `1 ≤ arr.size() ≤ 10⁶`  
- `0 ≤ arr[i] ≤ 10⁶`
